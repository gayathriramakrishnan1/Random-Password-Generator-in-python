# 🔐 Random Password Generator (Python)

## 📌 Description
This project is a Python-based random password generator that creates secure and strong passwords using a combination of letters, digits, and special characters. It ensures better password strength by including at least one character from each category.

---

## 🎯 Objective
- To generate secure passwords for user accounts  
- To demonstrate the use of Python libraries like `random` and `string`  
- To understand basic input handling and error management  

---

## 🚀 Features
- Generates strong random passwords  
- Includes uppercase & lowercase letters  
- Includes numbers and special characters  
- Ensures minimum complexity (letters, digits, symbols)  
- User-defined password length  
- Input validation for incorrect entries  

---

## 🛠️ Technologies Used
- Python  
- Built-in Libraries:
  - `random`
  - `string`
